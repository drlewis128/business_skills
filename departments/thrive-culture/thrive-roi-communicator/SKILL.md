# THRIVE ROI Communicator

The ROI Communicator trains teams to speak in the language that drives MBL decisions: measurable value, dollar terms, and BLUF-first delivery. This skill helps managers, entity CEOs, and individual contributors articulate the value they create in the format that Matt Mathison, Dr. Lewis, and investors actually use — cutting through effort narratives and activity reports to get to the number that matters.

## When to Use

- A manager or team member needs to present their work's value to leadership in an ops meeting, review, or update
- An entity CEO is preparing the ROI section of a portfolio ops deck or LP briefing
- A team has created real value but is communicating it in a way that sounds like an activity report
- Dr. Lewis is coaching an entity CEO on how to frame business performance for Matt Mathison

## ROI Communicator Framework

```
THRIVE ROI COMMUNICATOR FRAMEWORK
====================================

SECTION 1 — THE MBL ROI COMMUNICATION STANDARD
  Lead with the number: "We created $X in value this quarter"
  Then: how (the mechanism), over what period, versus what alternative
  Never lead with the effort: "We worked hard on X, and as a result..."
  Matt Mathison's "so what?" test: if the first sentence doesn't answer "so what?", the communication fails
  Dr. Lewis standard: every update has a value claim; every value claim has a number

SECTION 2 — INVESTOR MINDSET COMMUNICATION MODEL
  Frame every communication as if presenting to an investor who is deciding whether to allocate more capital
  Investor questions to answer:
    1. How much value was created? (dollar or %)
    2. Over what period?
    3. With what investment (cost, time, headcount)?
    4. What is the ROI (value / investment)?
    5. What is the trajectory (improving / stable / declining)?
    6. What is the next value creation opportunity and expected return?
  This framework applies to: initiative updates, ops review reports, 1:1 updates, LP communications, client briefings

SECTION 3 — BLUF-FIRST VALUE REPORTING
  Bottom line: [Value created] — [dollar amount or %] — [period]
  Supporting evidence: mechanism, data source, comparison to baseline
  Context: what changed, what was invested, what comes next
  Do NOT: lead with context; lead with effort; lead with caveats; bury the number
  Matt Mathison format: "We [did X]. It produced $[Y] in [category]. Here's how."

SECTION 4 — ENTITY CEO ROI BRIEFINGS
  Entity CEOs brief Dr. Lewis monthly; Dr. Lewis briefs Matt Mathison monthly
  Entity CEO briefing format: 1 page; top 3 value events; entity-level metrics vs. targets; next 30-day value commitment
  Allevio CEO: PMPM trend + denial rate + member count + revenue vs. plan
  HIVE CEO:    BOE vs. plan + lifting cost + LP distribution status + WTI hedge position
  Column6 CEO: gross margin % + IO execution rate + client LTV trend + pipeline value
  MBL CEO:     Portfolio value creation ($) + AI systems ROI + deal pipeline value

SECTION 5 — MATT MATHISON ROI SUMMARY FORMAT
  One page, dollar terms, specific — no approximations, no ranges without explanation
  Structure: [Entity] → [Value Created This Period: $X] → [Top Driver] → [Next Commitment]
  Frequency: monthly from each entity CEO; quarterly portfolio rollup
  Tone: direct, factual, investor-grade — not enthusiastic, not deferential
  He interprets: vague language = low value; hedged numbers = data quality problem; no number = no accountability

SECTION 6 — FRAMING QUALITATIVE VALUE IN QUANTITATIVE TERMS
  Common qualitative claims and how to translate them:
    "Improved client relationships" → estimated revenue at risk from churn × probability reduction
    "Built team capability" → productivity gain estimate × hours saved × loaded cost
    "Increased brand awareness" → pipeline influenced × average conversion value
    "Improved culture" → voluntary turnover reduction × cost-per-hire × headcount affected
    "Reduced risk" → estimated cost of the risk event × reduction in probability
  If translation is not possible: be honest — "This has qualitative value I cannot yet quantify. Here is my plan to measure it."

SECTION 7 — VALUE COMMUNICATION TRAINING FOR MANAGERS
  Common failure modes:
    Activity report: "We completed X deliverables and held Y meetings" — no value claim
    Effort narrative: "The team worked really hard and put in extra hours" — no value claim
    Causal ambiguity: "Revenue went up. We worked on client relationships." — no clear connection
    Precision theater: "We improved efficiency by approximately 12-15% in some areas" — not credible
  Training exercise: take any status update; apply the investor mindset questions; rewrite
  Manager accountability: if their update doesn't pass the "so what?" test, they own the rewrite
```

## Output Contract

- **The BLUF-first format is enforced because buried value claims lose in fast-moving leadership contexts.** Matt Mathison reviews multiple entity updates in a compressed window. An update that reaches its value claim in paragraph three has already failed — not because the value isn't there, but because the communication didn't respect the reader's time or decision-making context. BLUF-first is a structural respect act aligned with THRIVE's Respect pillar applied to communication.

- **Qualitative-to-quantitative translation is required — with explicit honesty when it cannot yet be done.** The alternative (accepting qualitative claims without translation) creates a two-tier accountability system where people doing measurable work are held to the number and people doing unmeasurable work are not held to anything. The translation requirement levels the accountability plane. The honest exception ("I cannot yet quantify this — here is my measurement plan") is acceptable; the vague exception ("this has significant but difficult-to-quantify value") is not.

- **The Matt Mathison ROI summary format is not a template preference — it is a decision support standard.** Matt Mathison allocates capital, time, and attention across a portfolio. The format he uses enables direct comparison across entities and over time. Deviation from the format (ranges instead of numbers, qualitative claims instead of dollar terms, multi-page updates instead of one-page) degrades his ability to make fast, accurate decisions. The skill enforces format compliance because the format serves the decision-maker's needs, not the presenter's comfort.

## Test Cases

1. **Golden path:** Allevio entity CEO needs to brief Dr. Lewis on Q2 value creation. Skill pulls PMPM improvement, denial rate reduction, and member count growth, translates each to dollar impact, and generates a 1-page BLUF-first briefing with $2.1M in total value created, three top events, and next-30-day commitment. Dr. Lewis reviews and routes to Matt Mathison portfolio rollup.

2. **Edge case:** A HIVE team member wants to communicate the value of a safety training program, which is entirely qualitative. Skill applies the risk-reduction translation framework: training covered 28 field workers; industry data shows similar training reduces OSHA-recordable incidents by 35%; HIVE's average incident cost is $45K; expected annual risk reduction value = 35% × historical incident rate × $45K per incident = $[X]. The translation gives a defensible, conservative dollar figure for what would otherwise be a qualitative claim.

3. **Adversarial:** A Column6 manager submits an ROI update claiming $800K in "brand value created" from a new content program with no quantification methodology and no connection to revenue or pipeline data. Skill flags the claim as precision theater, returns the update with the investor mindset questions applied, requires the manager to either provide a quantification methodology or reclassify the claim as "qualitative value — measurement plan pending," and logs the submission for entity CEO awareness.

## Audit Log

All ROI communications generated, entity CEO briefings reviewed, and Matt Mathison submissions are logged with entity, period, total value claimed, dollar methodology used, and reviewer approvals. Claims that were revised following "so what?" challenge are logged with original and revised amounts. Logs support quarterly value creation trend analysis and LP governance reporting.
